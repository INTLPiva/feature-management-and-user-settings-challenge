# Feature Management & User Settings

Sistema completo de gerenciamento de preferências do usuário e feature flags, desenvolvido com Node.js + PostgreSQL no backend e React Native + Expo no frontend.

## 📋 Sobre o Projeto

Este projeto implementa um sistema de configurações de usuário com feature flags dinâmicas, permitindo habilitar/desabilitar funcionalidades remotamente. Desenvolvido seguindo os princípios de Clean Architecture no backend e MVVM no frontend.

## 🏗️ Arquitetura

- **Backend**: Node.js, TypeScript, Fastify, PostgreSQL (Clean Architecture)
- **Frontend**: React Native, Expo, TypeScript (MVVM)

## 📁 Estrutura do Repositório

```
.
├── backend/          # API REST em Node.js
└── frontend/         # App móvel em React Native
```
